# US-06: Trading Session - Setup & Connection

**Priority:** High (MVP Core)

**As a** User,
**I want** to easily connect with another user's device,
**So that** we can view each other's binders.

## Acceptance Criteria
- [ ] **AC1:** User A can "Start Session" -> Generates a unique short code (e.g., "AF3D").
- [ ] **AC2:** User A sees a QR code representing that session link.
- [ ] **AC3:** User B can "Join Session" by typing the code.
- [ ] **AC4:** Both users land on a shared "Trade Room" interface.
