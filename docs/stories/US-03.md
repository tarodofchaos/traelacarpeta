# US-03: User Registration & Authentication

**Priority:** High (MVP Core)
**Status:** Complete

**As a** User,
**I want** to sign up and log in using my email and password,
**So that** I can securely access my personal binder.

## Acceptance Criteria
- [x] **AC1:** User can register with Email/Password.
- [x] **AC2:** User can login and receive a session token (JWT or Session).
- [x] **AC3:** Passwords are hashed (bcrypt/argon2).
- [x] **AC4:** Protected routes require authentication.
