# US-03: User Registration & Authentication

**Priority:** High (MVP Core)

**As a** User,
**I want** to sign up and log in using my email and password,
**So that** I can securely access my personal binder.

## Acceptance Criteria
- [ ] **AC1:** User can register with Email/Password.
- [ ] **AC2:** User can login and receive a session token (JWT or Session).
- [ ] **AC3:** Passwords are hashed (bcrypt/argon2).
- [ ] **AC4:** Protected routes require authentication.
