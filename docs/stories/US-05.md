# US-05: Manage Wants List

**Priority:** High (MVP Core)

**As a** User,
**I want** to maintain a list of cards I am looking for,
**So that** trading partners know what to offer me.

## Acceptance Criteria
- [ ] **AC1:** Search bar allows adding cards to "Wants".
- [ ] **AC2:** User can specify "Any Version" vs "Specific Version".
- [ ] **AC3:** User can view their Wants list.
