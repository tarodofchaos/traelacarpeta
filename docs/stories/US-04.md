# US-04: Manage Collection (Virtual Binder)

**Priority:** High (MVP Core)
**Status:** Complete (Backend API)

**As a** User,
**I want** to add, view, and remove cards in my digital binder,
**So that** I can track what I have available for trade.

## Acceptance Criteria
- [x] **AC1:** Search bar allows finding cards by name from the local `Card` database.
- [x] **AC2:** "Add Card" modal captures: Condition, Language, Finish, Quantity.
- [x] **AC3:** User can view a list/grid of their collection.
- [x] **AC4:** User can edit quantity or delete an entry.
