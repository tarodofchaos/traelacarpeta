# US-04: Manage Collection (Virtual Binder)

**Priority:** High (MVP Core)

**As a** User,
**I want** to add, view, and remove cards in my digital binder,
**So that** I can track what I have available for trade.

## Acceptance Criteria
- [ ] **AC1:** Search bar allows finding cards by name from the local `Card` database.
- [ ] **AC2:** "Add Card" modal captures: Condition, Language, Finish, Quantity.
- [ ] **AC3:** User can view a list/grid of their collection.
- [ ] **AC4:** User can edit quantity or delete an entry.
